# 🎌 Anime Recommendation by Mood – MVP Plan

A web app that recommends anime based on your mood or categories. Think: *“What should I watch if I feel nostalgic?”* or *“Give me an underrated isekai.”*

---

## ✅ MVP Features

- 🎭 **Mood picker** (e.g., happy, sad, nostalgic, adrenaline)
- 🗂️ **Category filters** (slice of life, isekai, horror, romantic, etc.)
- 🔄 **Random recommendation** with filters
- 📺 **Embed trailers** or anime cards with basic info
- 🔗 **Streaming links** (Crunchyroll, Netflix, etc. via JustWatch API)
- ⭐ “If you liked X, try Y” suggestions
- 🌙 **Dark/Light mode**
- 🕹️ Responsive mobile-friendly design

---

## ⚖️ Pros & Cons

### ✅ Pros

- Solves a common pain point: *“What anime should I watch?”*
- High search volume keywords (great for SEO)
- Evergreen content with constant updates
- Easily expandable with more features (watchlists, quizzes, AI picks)
- Potential for community interaction or gamified suggestions
- Monetizable via affiliate links or Pro features

### ⚠️ Cons

- Streaming availability requires a third-party API or scraping
- Mood tagging anime accurately takes effort (manual or ML)
- Competition with existing platforms like AniList, MAL, and Reddit subs
- Need to keep anime data fresh (airing/completed statuses)

---

## 🧠 Suggested Tech Stack

- **Frontend**: HTML + SCSS + Alpine.js or Vue/Nuxt
- **Backend**: PHP or Supabase (if needed)
- **APIs**:
    - AniList (anime metadata)
    - JustWatch or WatchMode (streaming availability)
